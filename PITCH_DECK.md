## Design Notes for Visual Creation

### Color Palette Usage:
- **Primary backgrounds:** Use Adenine (#1A2F3D) for major sections
- **Accents:** Use Thymine (#2A5A5B) for CTAs and highlights
- **Highlights:** Use Cytosine (#B8935E) for key metrics and important info
- **Secondary:** Use Guanine (#4A5568) for supporting elements

### Typography:
- **Headlines:** Large, bold, serif font (Freight Display Pro style)
- **Body:** Clean sans-serif (Acumin Pro style)
- **Hierarchy:** Clear size differentiation

### Visual Style:
- Clean, minimal design
- DNA-inspired elements (subtle, not overwhelming)
- Professional photography where appropriate
- Charts and graphs for metrics
- Icons for services and benefits

### Slide Layout:
- Generous white space
- Consistent margins and padding
- One main idea per slide
- Visual hierarchy guides the eye

---

## Presentation Tips

1. **Opening:** Start with the problem (Slide 2) to create urgency
2. **Story Arc:** Problem → Solution → Framework → Services → Results → CTA
3. **Timing:** 15-20 minutes for full deck, 5-7 minutes for elevator pitch
4. **Customization:** Adapt slides based on audience (investors vs. potential clients)
5. **Visuals:** Use actual screenshots, metrics, and case studies when available
